邮件处理功能

功能概述
- 通过 IMAP 拉取邮件，仅保存非广告邮件到知识库 `data/kb/<id>/`。
- 使用 Claude Code 进行广告判定；广告邮件直接丢弃不入库。
- 支持邮箱白名单（环境变量 `MAIL_WHITELIST`），白名单发件人跳过广告判定，直接总结入库。
- 为每封入库邮件生成 `summary.md` 摘要。
- 对可解析的附件生成同名 Markdown 摘要 `{附件}.md`，与附件存放在同一目录。

环境变量
- `MAIL_WHITELIST`: 逗号分隔的邮箱地址，大小写不敏感
- 其他必要变量：`MAIL_USER`、`MAIL_PASS`、`ANTHROPIC_*` 等

目录结构
- `data/kb/<ISO-uid>/email.json`
- `data/kb/<ISO-uid>/summary.md`
- `data/kb/<ISO-uid>/<附件>` 与 `<附件>.md`

验证
- 运行 `npm test` 执行单元测试
